## Perfil
### Processo de cadastro de usuário

1. O usuário preenche os dados básicos (nome, email, senha, confirmação de senha)
2. O status do usuário é definido como "pendente"
3. O usuário confirma o cadastro e é redirecionado para a página de login
4. O usuário faz login na aplicação e é redirecionado para a página inicial
5. Na tela de home exibe um popup para terminar o cadastro
5.1 O usuário informa se é barbeiro ou barbearia
5.2 Se for barbearia, o usuário preenche cnpj
5.3 Se for barbeiro, o usuário preenche cpf
5.4 O sistema valida o cpf/cnpj informado
5.5 O usuário preenche os dados de contato (instagram, telefone)
5.6 O usuário preenche os dados de endereço (cep, estado, cidade, bairro, rua, número e complemento)
5.7 O usuário informa se atende in loco
6. O usuário confirma o cadastro e é redirecionado para a página inicial

### Configuração do usuário

1. O usuário define o horário de funcionamento (abertura e fechamento)
2. O usuário define o intervalo de tempo entre agendamentos (em minutos)
3. O usuário define os serviços que oferece (barba, corte, coloração, etc.)
4. O usuário define a comissão por serviço e por profissional
5. O usuário define seus profissionais (apenas se barbearia)
